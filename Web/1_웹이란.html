웹 공부

이전에

네트워크 - 정보 전달을 위해 두 대 이상의 기기를 연결해 서로 통신할 수 있게 해주는 것

인터넷 - 여러 통신망을 하나로 연결한다는 의미 - international network

전 세계의 모든 컴퓨터들이 연결된 전 세계적인 네트워크

TCP, 패킷, 작동원리 등 더 딥한 개념은 공부 필요

웹이란

www or w3- world wide web의 줄임말

인터넷에 연결된 전세계 사용자들이 서로의 정보를 공유 할 수 있는 공간

인터넷상에서 텍스트, 그림, 영상 등과 같은 멀티미디어 정보를 하이퍼텍스트 방식으로 연결하여 제공

하이퍼텍스트는 HTML로 표현됨

웹은 HTTP 프로토콜을 사용

웹은 인터넷 위에서 동작하는 수많은 서비스중 하나
